## 🚀 About The Project

AI News Bot is a powerful Telegram bot that fetches the latest AI and tech news from top RSS feeds and converts them into engaging, SEO-friendly Facebook captions using **Groq's LLaMA 3 AI model**.

### ✨ Features

- 🔄 **Real-time News Fetching** from TechCrunch, The Verge, MIT Tech Review, and Hacker News
- 🤖 **AI-Powered Content Generation** using Groq LLaMA 3
- 📝 **Agentic AI Format** with:
  - Scroll-stopping hooks
  - Natural engaging body
  - Benefits highlighting
  - Strong Call-to-Action
  - 5-8 relevant hashtags
- 🆕 **New + Old Stories** support (30 days retention)
- 🖼️ **Image Prompts** for each story
- 🌐 **Fully in Bangla** for Bengali audience
- 📱 **Facebook-Optimized** content

### 🛠️ Tech Stack

- **Python 3.8+**
- **python-telegram-bot** - Telegram Bot API
- **Groq API** - LLaMA 3 for content generation
- **RSS Feeds** - News aggregation
